This web applet takes a system of equations in the form of \( A\vec{x} = \vec{b} \) where \( A \) is an \( m \times n \) constant coefficient matrix, \( \vec{x} \) is a \( n \times 1 \) variable column vector, and \( \vec{b} \) is an \( m \times 1 \) constant coefficient column vector. Then, the app solves it using the gaussian elimination algorithim. To use this web app, select the number of variables and equations in the box to the left and input coeffiecients for the variables. Then press calculate!
