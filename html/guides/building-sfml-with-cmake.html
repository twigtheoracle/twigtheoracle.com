Building SFML With CMake ON WINDOWS


So I was having an absurd number of issues building SFML from the source code with CMake. I managed to figure it out after a few days, and I realized that the online guide does not do a good job of explaining things if you are a complete noob at compiling libraries and have no idea what CMake is. Therefore, you have this guide.

What do you need?

  • The source code of SFML. This can be found on their website under the download tab. Click on the top left button (Latest stable version), then download the Source Code.
    • Then, just extract the zip file into an easily accessible location, like "C:\" or your desktop
  • CMake GUI. Go to CMake's downloads page and download the proper version of CMake for your OS. Note that on Windows, x86 means 32 bit, x64 means 64 bit. If you don't know what your Windows install is, Google how to find out.
    • It is helpful but not necessary to add CMake to the system PATH variable. If you don't know what that is, try googling it :)

  • Some sort of C++ compiler. I suggest using TDM-GCC (because that's what I used and it works). Again, go to their downloads page and pick up whatever version you want. If you don't know and don't care, click on the first download link. If you don't know but do care, google is your friend.
    • BUT, if you have a compiler installed already, it's a-ok to use that one as well.
    • If you are installing TDM-GCC version whatever, just create a new install with all the default values

What do you need to do?

  1. Assuming that you have installed/downloaded everything properly, open up CMake. You will see a screen like the one to the right
  2. The first thing you want to do in CMake is find the source code.
